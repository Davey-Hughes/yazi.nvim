# A Neovim Plugin for [yazi](https://github.com/DreamMaoMao/yazi.nvim.git)

# install

```json
 {
  "DreamMaoMao/yazi.nvim",
  dependencies = {
    "nvim-telescope/telescope.nvim",
    "nvim-lua/plenary.nvim",
  },

  keys = {
    { "<leader>gy", "<cmd>Yazi<CR>", desc = "Toggle Yazi" },
  },
}
```

### Usage
```
:Yazi

or 

leader + gy
```
